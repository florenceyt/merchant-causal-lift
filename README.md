# merchant-causal-lift

**Heterogeneous treatment effect estimation for a merchant equipment offer using PSM + CausalForestDML.**

A payments company offered discounted card readers to a subset of new merchants. This pipeline estimates the incremental impact on 90-day payment volume — accounting for selection bias — and identifies which merchant segments benefit most.

---

## The Problem

Larger, more established merchants were preferentially targeted for the offer, making a naive treated-vs-control comparison misleading. This pipeline uses:

1. **Propensity Score Matching (PSM)** to create a balanced comparison group
2. **CausalForestDML** (EconML) to estimate heterogeneous treatment effects (CATE) at the merchant level
3. **Segment analysis** to surface high-ROI targeting priorities

---

## Results (Synthetic Data)

| Metric | Value |
|---|---|
| Matched pairs (post-PSM) | 1,174 |
| ATE estimate | ~$1,200 per merchant |
| Highest-lift segment | Restaurant (~$1,480) |
| Lowest-lift segment | Entertainment (~$850) |

**Key finding:** The effect is heterogeneous. Restaurants and Retail merchants respond ~75% more strongly than Entertainment merchants — a targeted rollout prioritizing these segments would meaningfully outperform a blanket campaign.

## Methodology

### Selection Bias
Merchants were not randomly assigned to receive the offer. Businesses with higher pre-period transaction volume, more locations, and existing POS systems were more likely to be targeted. A naive difference-in-means would confound merchant quality with offer impact.

### Stage 1 — Propensity Score Matching
A logistic regression estimates each merchant's probability of treatment given observed covariates. Merchants are matched 1:1 on the logit of the propensity score within a caliper of 0.05 log-odds units. Post-match balance is validated using standardized mean differences (SMD < 0.10 threshold).

### Stage 2 — CausalForestDML
EconML's `CausalForestDML` uses cross-fitted gradient boosting models to residualize both the outcome and treatment on covariates, then fits a Causal Forest on the residuals. This yields merchant-level CATE estimates with valid confidence intervals.

### Limitations
PSM removes observed confounding only. Unmeasured confounders (e.g., merchant growth motivation) cannot be eliminated without a randomized design. Results should be interpreted as observational estimates, not causal claims from an RCT.
